# Kotlin-Sample-App
A sample MVP Android application written in Kotlina that fetches news using NYT APIs.

Stack:

Kotlin
RxJava and RxAndroid
Retrofit / OkHttp
Gson
Dagger 2
Butterknife
Google Play Services
Timber
Glide 3
Stetho
Espresso for UI tests
Robolectric for framework specific unit tests
Mockito
